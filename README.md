# Bluetooh_Robot
this small project to develop a samall wheeled robot controlled using an android app via bluetooh 

## Component Used:
 ## Eelectronics Components:
  1. Arduino Uno
  2. LN298N H bridge Motor driver
  3. HC-05 Bluetooth Module
  4. 3s LiPo battery 
  5. 4x Motors 
## Chassis Harware
  1. Aluminium sheed 
  2. 4x Wheels
